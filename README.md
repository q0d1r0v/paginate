# paginate-by-oybek

## parameters

```
page => number

length => number

updatePage => method

```

## for example

```
<paginate :page="1" :length="27" @updatePage="goServer" />

methods: {
    goServer(pageNumber) {
      console.log(pageNumber)
    }
}

```

## first step =>
```
git clone https://github.com/qodirovoybekjon4929/paginate.git

```

### next step => 
```
cd paginate
```
